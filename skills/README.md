# Project Skills Index

These skills are project-local helpers for **gentle-landing**.

## Available Skills

| Skill | Use when... |
|---|---|
| `astro-ui-layout` | Editing Astro components, responsive layouts, hero sections, view transitions, or motion. |
| `i18n-copy-sync` | Working on English/Spanish copy, translation markers, or language-dependent wrapping. |
| `accessibility-a11y` | Adding buttons, tabs, keyboard navigation, ARIA labels, or focus behavior. |
| `responsive-visual-regression` | Fixing mobile/desktop layout issues, overflow, or visual stability regressions. |
| `content-integrity` | Verifying install commands, links, stats, docs, and upstream parity against `../gentle-ai`. |

## How to use

- Load the smallest skill that matches the task.
- Combine skills only when the task genuinely spans multiple concerns.
- Prefer these project-local skills over generic behavior when editing this repo.

## Notes

- The skills are intentionally small and focused.
- They are meant to keep the landing aligned with the real `../gentle-ai` project.

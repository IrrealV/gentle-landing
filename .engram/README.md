# Engram Snapshot — gentle-landing

This directory is a **project-only snapshot export** of Engram context for `gentle-landing`.

## Scope

- Contains only memories, decisions, notes, and summaries for this project.
- Does **not** expose the live Engram backend.
- Intended for people cloning the repo so they can understand the evolution of the landing.

## Files

- `context.md` — concise project context and current conventions
- `decisions.md` — important decisions and tradeoffs
- `timeline.md` — short chronological notes of major changes

## Updating

Regenerate this snapshot from the project's Engram memory when major decisions change.

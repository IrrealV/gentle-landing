# Proposal: Pixel-Perfect Stitch Design Rebuild

## Goal
Rebuild the Gentle-AI landing pages using Astro, ensuring a **1:1 match** with the provided `/stitch` HTML/CSS designs.

## Scope
- Foundation (Design Tokens, Typography, Layout)
- Shared Components (Navbar, Footer)
- UI Primitives (TerminalWindow, CodeTabs, Badge, Button)
- All pages (Index, HowItWorks, Configurator, Docs, Features)

## Approach
